# Telegram Mini App Deploy

## What Is Already Wired

- `@tonconnect/ui` is loaded in `apps/web/index.html`.
- `apps/web/app.js` initializes `TonConnectUI` with `/tonconnect-manifest.json`.
- `/api/config` exposes deployment config to the Mini App.
- `/tonconnect-manifest.json` is generated by the Node server from environment variables.
- Wallet page has:
  - connect / switch wallet
  - disconnect wallet
  - test payment of `0.01 TON` to `TON_TREASURY_ADDRESS`

## Required Environment

Copy `.env.example` into your deployment environment:

```bash
APP_NAME=TON Farm
APP_URL=https://your-mini-app-domain.example
APP_ICON_URL=https://your-mini-app-domain.example/icon.png
TON_NETWORK=mainnet
TON_TREASURY_ADDRESS=EQxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TELEGRAM_RETURN_URL=https://t.me/your_bot/your_app
```

`APP_URL` must be a public HTTPS URL. Telegram and wallet apps cannot use your local `http://localhost:4173` as the production Mini App URL.

## Deploy Web App

Any Node host is fine:

- Render
- Railway
- Fly.io
- VPS + nginx
- Cloudflare Tunnel for temporary testing

Run:

```bash
npm run start
```

Then verify:

```text
https://your-mini-app-domain.example/
https://your-mini-app-domain.example/api/config
https://your-mini-app-domain.example/tonconnect-manifest.json
```

## Configure BotFather

In Telegram:

1. Open `@BotFather`.
2. Use `/mybots` and select your bot.
3. Go to `Bot Settings`.
4. Configure `Menu Button` and set the HTTPS Mini App URL.
5. Go to `Configure Mini App`.
6. Enable the Mini App and set the same HTTPS Mini App URL.

After this, users can open the app from the bot profile or menu button.

## Current Chain Boundary

This repo now supports real wallet connection and real TON payment signing. NFT minting and marketplace settlement still need deployed contracts:

- `GameAssetCollection`
- `GameManager`
- `Marketplace`
- `Treasury`

Until those contracts are deployed and their addresses are added to the app, planting, harvest, inventory, and market actions remain backend-simulated game actions.
